# Мій перший HTML-сайт

Навчальний веб-проєкт, створений у межах дисципліни **“Створення WEB-сайтів (Front-End)”**.  
Сайт складається з трьох сторінок та демонструє базову структуру HTML5, роботу з формами і медіа.

## Реалізовано

### Головна сторінка (index.html)
- Семантична структура HTML5: header, nav, main, footer
- Заголовки, абзаци, списки
- Виділення тексту (strong, em)
- Зображення з підписом (figure, figcaption)
- Таблиця з даними
- Навігація між сторінками

### Сторінка контактів (pages/contact.html)
- Форма зворотного зв’язку
- Поля: ім’я, email, телефон, тема, повідомлення
- Radio-кнопки вибору способу зв’язку
- Checkbox згоди з правилами
- HTML5-валідація (required, minlength, pattern)

### Сторінка “Про мене” (pages/about.html)
- Коротка інформація про автора
- Вбудоване відео (video + source)
- Вбудоване аудіо (audio + source)
- Підписи та fallback-текст


## Як запустити

1. Завантажити або клонувати проєкт
2. Відкрити файл **index.html** у будь-якому браузері  
   (подвійний клік або через меню "Відкрити за допомогою")
