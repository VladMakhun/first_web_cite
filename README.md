# Мій перший веб-сайт (HTML + CSS)

## Опис проєкту

Це навчальний багатосторінковий веб-сайт, створений у рамках практичних робіт з HTML та CSS.
Сайт складається з трьох сторінок: головна, "Про мене" та "Контакти".

У роботі реалізовано базову структуру HTML5 та оформлення за допомогою CSS.

---

## Що було реалізовано

### HTML:

* Семантична структура: header, nav, main, footer
* Навігація між сторінками
* Заголовки, абзаци, списки (ul, ol)
* Зображення з підписом (figure, figcaption)
* Таблиця з даними
* Форма зворотного зв’язку (input, select, textarea, radio, checkbox)
* Вбудовані медіа: відео та аудіо

---

### CSS:

* Підключення зовнішніх стилів (style.css, responsive.css)
* Використання змінних (:root) для кольорової палітри
* Оформлення:

  * header та footer
  * навігаційного меню (Flexbox)
  * тексту та списків
  * карток (card)
  * таблиці
  * форми (input, кнопки, focus, hover)
  * зображень, відео та аудіо
* Використання Flexbox (меню + блок two-column)
* Реалізація адаптивності через media queries

## Як запустити проєкт

### Варіант 1 (рекомендовано):

1. Відкрити проєкт у Visual Studio Code
2. Встановити розширення **Live Server**
3. Натиснути правою кнопкою на `index.html`
4. Обрати **Open with Live Server**

---

### Варіант 2:

* Відкрити файл `index.html` подвійним кліком у браузері

---

## Адаптивність

Сайт адаптується під мобільні пристрої (до 768px):

* меню стає вертикальним
* блоки розташовуються в один стовпець
* зменшуються відступи та розміри тексту
