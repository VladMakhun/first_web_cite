#  Мій перший веб-сайт (HTML + CSS + JavaScript)

##  Опис
Це навчальний багатосторінковий веб-сайт, створений у рамках практичних робіт з дисципліни Front-End розробки.

Сайт демонструє базові навички роботи з:
- HTML5 (структура сторінок)
- CSS3 (стилізація та адаптивність)
- JavaScript (інтерактивність)

---

## Реалізований функціонал

###  HTML + CSS
- Семантична структура сторінок
- Навігаційне меню
- Картки (card layout)
- Таблиці
- Форми
- Адаптивна верстка (mobile-friendly)

---

###  JavaScript

####  Перемикач теми
- Світла / темна тема
- Збереження вибору в `localStorage`

####  Кнопка "Вгору"
- З’являється при прокрутці
- Плавна прокрутка до початку сторінки

####  Автоматичний рік
- Динамічно оновлюється у footer

####  Акордеон (About page)
- Розгортання / згортання блоку "Мої навички"

#### Форма зворотного зв’язку
- Валідація полів:
  - Ім’я
  - Email
  - Повідомлення
- Збереження введених даних (чернетка) в `localStorage`
- Обробка форми через JavaScript (`FormData`)
